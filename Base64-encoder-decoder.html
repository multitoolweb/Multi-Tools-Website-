About Base64 Encoding

Base64 is a binary-to-text encoding scheme that represents binary data in an ASCII string format. It's commonly used to embed binary data in text-based formats like HTML, CSS, and JSON.

Common Uses

  • Embed images in HTML/CSS without separate files
  • Encode file attachments in emails
  • Store binary data in databases as text
  • API data transfer for binary content
  • Data URLs for web resources

How It Works

Base64 converts every 3 bytes (24 bits) of binary data into 4 ASCII characters from a set of 64 characters (A-Z, a-z, 0-9, +, /).
