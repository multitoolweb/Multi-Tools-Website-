Features

  • Ignore Case: Treat uppercase and lowercase as same (e.g., "Apple" = "apple")
  • Trim Whitespace: Automatically remove spaces from start/end of each line
  • Sort Alphabetically: Arrange unique lines in alphabetical order
  • Keep Empty Lines: Option to preserve blank lines in output
  • Real-time Statistics: See counts for original, unique, and removed lines
  • One-click Copy: Copy cleaned text directly to clipboard
